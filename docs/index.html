常见问题

软件收费吗？
完全免费开源。基础对话调用的是文心一言海量语录库。大模型对话功能由于接入的是你自己的 API Key，有很多免费的api可以接入。戳这里看教程
支持 Mac 或 Linux 吗？
目前主打 Windows 体验，但由于基于 Tauri 框架，你可以克隆源码自行在 Mac 下编译运行。 因为主包对Mac和Linux不熟悉，所以暂时没有提供官方安装包。欢迎朋友贡献跨平台版本！ 可以通过我的抖音联系我dy1nrbj3gm4gd或者直接提交 PR！
怎么更换自己喜欢的宠物？
在网页上方前往开源社区下载 .zip 格式的宠物包，右键点击桌面上的宠物，进入设置，导入宠物即可。戳这里直达教程
