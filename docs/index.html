FAQ

常见问题

软件收费吗？

LingoPet项目完全开源，基础功能可直接使用，可以直接下载软件安装包，或者下载源码在此基础上进行二创，软件遵守MIT许可证。

macOS安装后 提示“已损坏，无法打开”？

因为 LingoPet 应用尚未签名和 Apple 公证。解决方法：将 LingoPet.app 拖到“应用程序”后，在终端执行下面的命令，然后再次打开应用即可解决。

Terminal 
xattr -cr /Applications/LingoPet.app
如何下载并更换喜欢的桌宠？

在桌宠管理-->宠物市场-->下载喜欢的宠物，即可召唤宠物（这里的宠物市场是codex宠物社区，兼容所有基础动作）

安装个性动作，可以在创意工坊中搜索。也可以桌宠管理-->我的桌宠-->编辑-->手动添加功德模式、专注模式、律动模式动作，创作的个性化动作也可以分享到创意工坊，供别的朋友使用。

如何制作桌宠个性化动作？

在桌宠管理-->我的桌宠-->编辑，打开编辑界面，复制待机动作的第一帧画面，去生图模型，编写提示词生成横版动作精灵表。

如果不是标准透明背景，可以在LingoPet Sprite Studio去除背景后，导入到软件编辑器中预览。

为什么说它很轻量？

前端使用 Vite + TypeScript + 纯 DOM，后端是 Tauri v2 + Rust，没有引入 Canvas/WebGL 或大型前端框架，桌宠窗口也保持小尺寸透明渲染。
